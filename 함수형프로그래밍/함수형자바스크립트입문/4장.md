# 클로저
- 간단히 얘기하면 **내부함수**다.

```javascript
function outer() {
  function inner() {

  }
}
```

위 `inner` 함수를 클로저 함수라고 부른다.

클로저는 **스코프 사슬(스코프 레벨)** 에 접근할 수 있어 유용하다.

클로저는 3개의 스코프를 갖고 있다. 
1. 자체 선언 내에서 선언된 변수
2. 전역 변수에 접근
3. 외부 함수(클로저 함수를 감싸는 함수)의 변수(파라미터 포함)에 접근(**중요!**)

자바스크립트 runtime 은 내부 함수가 반환될 때 위 3개의 스코프를 지정한다.
